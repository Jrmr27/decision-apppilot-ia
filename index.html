Decision Pilot IA

Miniaplicación educativa para estructurar decisiones complejas con ayuda de IA. Define el contexto, la decisión, los datos y las restricciones para generar un prompt razonado y reutilizable.

Selecciona un caso, revisa o completa los datos sugeridos y genera un prompt estructurado para utilizarlo después en una IA externa.

Configurar caso de decisión

Selecciona uno de los cuatro casos para cargar un ejemplo precargado.
